# Prototipo Plano de Saude Digital

Prototipo visual (nao funcional) de plataforma de plano de saude digital para jovens de 16+ anos.

## Versoes Disponiveis

| Arquivo | Marca | Cores |
|---------|-------|-------|
| `yovida-turquesa.html` | YoVida | Turquesa/Verde |
| `yovida-roxo.html` | YoVida | Roxo/Turquesa |
| `virei-multicolor.html` | Vire+ | Gradiente colorido |
| `virei-laranja.html` | Vire+ | Laranja/Roxo |

## Como Visualizar

1. Abra `index.html` no navegador
2. Escolha uma das 4 versoes de cores

## Funcionalidades do Prototipo

- Landing page com hero, planos e FAQ
- Mock de celular com navegacao mobile
- Telas: Login, Dashboard, Telemedicina, Pontos, Perfil
- Transicoes suaves entre telas
- Layout responsivo

## Estrutura

```
prototipo-plano/
├── index.html              # Pagina de selecao
├── yovida-turquesa.html    # Versao turquesa
├── yovida-roxo.html        # Versao roxa
├── virei-multicolor.html   # Versao multicolorida
├── virei-laranja.html      # Versao laranja
└── README.md
```

## Tecnologias

- HTML5
- CSS3 (sem frameworks)
- JavaScript vanilla
